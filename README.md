## 🚀 Getting Started
Setting up your own instance of Monvana Bank is simple.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/omjvibe/monvana.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Copy `env.example.txt` to `.env.local` and fill in your Supabase and Resend credentials.

4. **Run the development server**:
   ```bash
   npm run dev
   ```
